# Benötigte Bilder für Casa Eltinger Hof Website

Bitte laden Sie die folgenden Bilder in die entsprechenden Ordner hoch.

## Empfohlene Bildformate
- **Format**: JPG oder WebP (für beste Performance)
- **Auflösung**: Mindestens 1920x1080 Pixel für Hero-Bilder
- **Dateigröße**: Optimiert unter 500KB pro Bild (nutzen Sie Tools wie TinyPNG)

---

## Hauptbilder (Root-Ordner: /public/images/)

### Hero-Bild
- **Dateiname**: `hero.jpg`
- **Verwendung**: Vollbild-Hintergrundbild auf der Startseite
- **Empfehlung**: Außenansicht des Restaurants oder stimmungsvolles Innenraumbild
- **Mindestgröße**: 1920x1080 Pixel (am besten 2560x1440)

### Restaurant Innenansicht
- **Dateiname**: `restaurant-interior.jpg`
- **Verwendung**: Über Uns Sektion auf der Startseite
- **Empfehlung**: Einladende Atmosphäre, gute Beleuchtung

### Speisekarten-Bild
- **Dateiname**: `menu-featured.jpg`
- **Verwendung**: Speisekarten-Vorschau auf der Startseite
- **Empfehlung**: Appetitliches Gericht, z.B. Pasta oder Pizza

### OG-Image (Social Media)
- **Dateiname**: `og-image.jpg`
- **Verwendung**: Vorschaubild bei Social Media Shares
- **Größe**: Exakt 1200x630 Pixel

## Tipps für optimale Bilder

1. **Beleuchtung**: Verwenden Sie Bilder mit guter, warmer Beleuchtung
2. **Qualität**: Professionelle Fotos machen einen großen Unterschied
3. **Konsistenz**: Ähnlicher Stil und Farbgebung für alle Bilder
4. **Authentizität**: Echte Bilder aus Ihrem Restaurant wirken besser als Stock-Fotos
5. **Komprimierung**: Nutzen Sie https://tinypng.com/ um Dateigröße zu reduzieren
